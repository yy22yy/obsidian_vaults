# 一、寄存器体系

## 1. 通用寄存器

```text
AX BX CX DX

扩展寄存器

EAX EBX ECX EDX
```

功能记忆：

|寄存器|常见用途|
|---|---|
|AX|累加器|
|BX|基址寄存器|
|CX|计数寄存器|
|DX|数据寄存器|

高低位：

AX = AH + AL  
BX = BH + BL  
CX = CH + CL  
DX = DH + DL

---

## 2. 指针寄存器

```text
SP  Stack Pointer  
BP  Base Pointer
```

作用：

SP → 栈顶地址  
BP → 访问栈中参数

---

## 3. 变址寄存器

```text
SI Source Index  
DI Destination Index
```

常用于

字符串操作  
数组访问

---

## 4. 指令指针寄存器

IP / EIP

```text
作用：存储下一条指令地址
注意：不是通用寄存器
```


# 移位指令

## 逻辑移位

```text
SHL 左移  
SHR 右移
移出位丢弃
```

---

## 循环移位

```text
ROL 左循环  
ROR 右循环
特点:移出位回到另一端
10110001  
ROR 2
结果:01101100
```
