## 代码方式 vs 官网网页输入

| 对比项      | 代码调用 (API)                                          | 官网网页 (ChatGPT/Kimi 网页版) |
| -------- | --------------------------------------------------- | ----------------------- |
| **参数控制** | ✅ 完全可控：temperature、max_tokens、top_p、system prompt 等 | ❌ 固定或有限选项               |
| **批量处理** | ✅ 可自动化处理大量请求                                        | ❌ 手动一条条输入               |
| **集成能力** | ✅ 可嵌入应用、工作流                                         | ❌ 独立使用                  |
| **成本**   | 按 token 计费，通常更便宜                                    | 可能更贵或有使用限制              |
| **体验**   | 需要编程                                                | 即开即用                    |
