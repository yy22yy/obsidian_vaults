### 预训练到底学到了什么

这是最容易被误解的地方。预训练不是简单地"记住文本"，而是通过预测任务**被迫学习语言的深层结构**。

BERT的掩码任务为例：

```
训练数据：Wikipedia的一句话
"牛顿在[MASK]提出了万有引力定律"

要答对，模型必须同时学会：
  语法层面 → "在"后面接地点
  语义层面 → 牛顿是物理学家
  世界知识 → 万有引力与苹果落地的故事发生在英国
  逻辑层面 → 地点和事件的合理对应关系

答案：[MASK] = "英国"
```

**一个预测任务，逼迫模型学习了语法、语义、知识、逻辑四个层面。** 这些能力不是被显式教导的，而是为了完成任务自然涌现的。

预训练结束后，模型的参数中隐含地存储了：

```
语言能力：
  语法结构、词语搭配、句式规律

语义能力：
  词义消歧、指代关系、情感倾向

世界知识：
  实体关系、常识推理、因果逻辑

表示能力：
  把任意文本映射成有意义的向量
```
