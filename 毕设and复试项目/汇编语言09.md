# 汇编语言基础选择题解析

## 1. 下面不是通用寄存器组中的寄存器

题目  
下面 **不是通用寄存器组中的寄存器**。

- A AL
- B EBX
- C EIP
- D BP

**答案：C**

### 解析

x86寄存器可分为：

| 类别            | 寄存器             |                        |
| ------------- | --------------- | ---------------------- |
| 通用寄存器         | AX BX CX DX     | 以X结尾，谐音register--X     |
| 扩展通用寄存器       | EAX EBX ECX EDX | 多加了E                   |
| 指针寄存器         | SP BP           | Segement/Basic Pointer |
| 变址寄存器         | SI DI           |                        |
| 指令寄存器（非通用寄存器） | IP / EIP        | Instruction Pointer    |

其中

- **AL**：AX的低8位
- **EBX**：32位通用寄存器
- **BP**：基址寄存器
- **EIP**：**指令指针寄存器**

EIP用于存储**下一条指令地址**，不属于通用寄存器。

### 知识点

#### x86寄存器分类

**通用寄存器**

```
AX BX CX DX
```

**扩展寄存器**

```
EAX EBX ECX EDX
```

**指针寄存器**

```
SP  Stack Pointer
BP  Base Pointer
```

**变址寄存器**

```
SI  Source Index
DI  Destination Index
```

**控制寄存器**

```
IP / EIP
```

# 2. POP 指令对 ESP 的影响

题目

已知

```
(ESP) = 100H
```

执行

```
POP BP
```

之后

```
(ESP) = ?
```

选项

- A 0FEH
- B 0FCH
- C 104H
- D 102H

**答案：D**

### 解析

`POP` 指令：

```
POP reg
```

执行过程

```
reg ← [ESP]
ESP ← ESP + operand_size
```

因为

```
BP = 16bit = 2字节
```

所以

```
ESP = 100H + 2
     = 102H
```

### 知识点

#### PUSH / POP 栈变化

**PUSH**

```
ESP = ESP - size
[ESP] = data
```

**POP**

```
data = [ESP]
ESP = ESP + size
```

|指令|ESP变化|
|---|---|
|PUSH WORD|ESP - 2|
|POP WORD|ESP + 2|
|PUSH DWORD|ESP - 4|
|POP DWORD|ESP + 4|

# 3. C语言变量在汇编中的定义

题目

C语言

```
char FLAG = 1;
```

汇编等价为

- A MOV FLAG,1
    
- B FLAG DB 1
    
- C FLAG EQU 1
    
- D FLAG = 1
    

**答案：B**

### 解析

C代码含义

```
定义一个1字节变量 FLAG
初值 = 1
```

汇编中

```
FLAG DB 1
```

含义

```
FLAG : 定义字节变量
DB   : Define Byte
1    : 初值
```

其他选项

**A**

```
MOV FLAG,1
```

只是赋值，不是变量定义。

**C**

```
EQU
```

符号常量。

**D**

```
=
```

|伪指令|含义|
|---|---|
|DB|定义字节|
|DW|定义字|
|DD|定义双字|

```
A DB 10
B DW 100
C DD 1000
```

# 4. 符号常量特征

**不属于符号常量特征**

- A 汇编期间被代换
- B 指明数据类型
- C 提高可读性
- D 方便修改

**答案：B**

### 解析

符号常量

```
COUNT EQU 10
```

特征

- **汇编期替换**
- **没有存储空间**
- **没有数据类型**

因此

```
B 错误
```

### 知识点

#### 符号常量

定义

```
NAME EQU value
```

例

```
MAX EQU 100
```

编译后

```
MOV AX,MAX
```

变成

```
MOV AX,100
```

特点

|特点|说明|
|---|---|
|无存储空间|只是替换|
|汇编期替换|类似宏|
|无类型|纯数值|

# 5. ROR 循环右移

```
DX = 0FFF0H
ROR DX,4
```

求结果。

选项

- A 0FFFFH
- B 0FF00H
- C 0F0FFH
- D 0FFFH

**答案：C**

### 解析

DX

```
0FFF0H
```

二进制

```
0000 1111 1111 1111 0000
```

但 DX 是 **16位**

所以实际

```
1111 1111 1111 0000
```

右循环4位

```
1111 0000 1111 1111
```

转换为16进制

```
F0FFH
```

即

```
0F0FFH
```

### 知识点


```
10110001
ROR 2

注意寄存器的有效位数
DX--16位

```

# 7. PUBLIC 声明

将 BUF1 BUF2 声明为公共符号

- A PUBLIC BUF1:WORD,BUF2:WORD
- B EXTRN BUF1,BUF2
- C PUBLIC BUF1,BUF2
- D PUBLIC BUF1 BUF2

**答案：C**

### 解析

PUBLIC 用法

```
PUBLIC symbol1,symbol2
```

正确

```
PUBLIC BUF1,BUF2
```

**EXTRN**

表示外部符号。

### 知识点

模块链接

|关键字|作用|
|---|---|
|PUBLIC|本模块导出|
|EXTRN|外部引用|

例

模块A

```
BUF1 DB 10
PUBLIC BUF1
```

模块B

```
EXTRN BUF1
```

# 变量 vs 常量（核心区别）



|类型|是否占内存|是否可修改|
|---|---|---|
|变量(DB)|占内存|可修改|
|EQU常量|不占内存|不可修改|



```cpp
变量

FLAG DB 1

- 内存结构

地址1000 → 01

- 程序中

MOV FLAG,2

可以修改。
```



``` cpp
符号常量

FLAG EQU 1

- 程序中

MOV AX,FLAG

- 汇编后

MOV AX,1

FLAG 根本不存在。
```
