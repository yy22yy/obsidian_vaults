Python的标准库--os
os 是 Python 的标准库，用于与**操作系统交互**：

| 常见用途    | 示例                                 |
| ------- | ---------------------------------- |
| 读取环境变量  | os.environ["HF_TOKEN"]             |
| 操作文件路径  | os.path.join("folder", "file.txt") |
| 创建/删除目录 | os.mkdir("new_folder")             |
| 执行系统命令  | os.system("ls")                    |
