[[预训练]]给模型"世界知识"，微调给模型"专业方向"。

### 微调到底在做什么

微调不是"重新学习"，而是**在已有能力上做方向性调整**。

一个直觉的理解是参数空间的导航：

```
预训练后，模型处于参数空间中的某个位置P
这个位置代表"通用语言理解能力"

              通用语言理解
                   P
                  /│\
                 / │ \
                /  │  \
         情感分析 问答 翻译  ← 微调把P轻推向目标方向
```

微调时发生的事：

```
冻结底层（保留通用能力）
  第1-8层：几乎不变  ← 语法、基础语义

轻微调整中层
  第9-10层：小幅变化 ← 任务相关的语义模式

重点调整顶层
  第11-12层：较大变化 ← 任务特定的决策逻辑

新增输出层
  分类头/生成头    ← 任务的具体输出形式
```

**底层是通用的地基，顶层是任务的装修。微调只改装修，不动地基。**

### 为什么这个范式如此重要---关于[[数据标注]]

在BERT之前，NLP领域面临一个根本困境：

**标注数据的稀缺性**

```
获取未标注文本：免费，无限
  → 整个互联网都是训练数据

获取标注数据：昂贵，稀缺
  → 需要专家人工标注
  → 医疗标注：需要医生
  → 法律标注：需要律师
  → 每个任务都要重新标注
```

预训练+微调彻底改变了这个局面：

```
预训练阶段：
  输入：数十亿无标注文本（免费）
  输出：强大的通用语言模型

微调阶段：
  输入：几百到几千条标注数据（可负担）
  输出：特定任务的专业模型
```

**把对标注数据的依赖降低了1000倍以上。**

### 知识迁移的层次结构

预训练学到的知识，在微调中的迁移是有层次的：

```
层次1：表层迁移（总是发生）
  词汇理解、基础语法
  → 任何任务都能用上

层次2：结构迁移（大多数任务）
  句子结构、语义组合规律
  → 文本分类、情感分析等

层次3：知识迁移（特定任务）
  世界知识、实体关系
  → 问答系统、信息抽取

层次4：推理迁移（高级任务）
  逻辑推理、因果关系
  → 阅读理解、自然语言推理
```

越底层的能力越容易迁移，越高层的能力越任务特定。

**就像加法和矩阵加法，知识也是有层级性质的**
