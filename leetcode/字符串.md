判断类：
==要善于用判断类==
```cpp
一般的判断方法

for(char c: s){
	if(!isdigit(c))//如果不是数字
	{return false;}
}
--在判断为全零之后直接用类型转换函数
int = stoi(s)
float = stof(s)
double = stob(s)
```


```cpp
---一个经典的例子
bool checkSegment(const string& seg) { 
	if (seg.empty()) return false; // 空段非法 
	if (seg.size() > 1 && seg[0] == '0')  return false; //前导0
	for (char c : seg) {if (!isdigit(c)) return false; }// 必须全是数字 
	int val = stoi(seg); 
	return val >= 0 && val <= 255; // 范围检查 
}
```

```cpp
for (char c : line) {
```

|函数|作用|返回非0为true|
|---|---|---|
|`isalpha(c)`|是字母|a-z A-Z|
|`isdigit(c)`|是数字|0-9|
|`isalnum(c)`|是字母或数字|a-z A-Z 0-9|
|`islower(c)`|是小写字母|a-z|
|`isupper(c)`|是大写字母|A-Z|
|`isspace(c)`|是空白字符|空格、`\t`、`\n` 等|

**转换类：**

|函数|作用|示例|
|---|---|---|
|`tolower(c)`|转小写|`'A'→'a'`，非字母原样返回|
|`toupper(c)`|转大写|`'a'→'A'`，非字母原样返回|

## 强制[[类型转换]] `(int)lower`

cpp

```cpp
char lower = 'a';
(int)lower  →  97      // 'a' 的 ASCII 值

seen[(int)lower] // 显式转换，意图清晰 
seen[lower] // 完全等价，编译器自动隐式转换
```

## 验证一下

cpp

```cpp
char c = 'A';
```
