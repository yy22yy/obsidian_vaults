# 四、机试高频模板

## ✅ BFS（最短路径）

```cpp
queue<int> q;
vector<bool> vis(n,false);
q.push(start);
vis[start]=true;

while(!q.empty()){
    int u=q.front(); q.pop();
    for(int v:adj[u]){
        if(!vis[v]){
            vis[v]=true;
            q.push(v);
        }
    }
}
```

---

## ✅ DFS（连通块）

```cpp
void dfs(int u){
    vis[u]=true;
    for(int v:adj[u]){
        if(!vis[v]) dfs(v);
    }
}
```

---

## ✅ 并查集（连通性）

```cpp
int fa[N];
int find(int x){
    return fa[x]==x ? x : fa[x]=find(fa[x]);
}
void merge(int a,int b){
    fa[find(a)]=find(b);
}
```
