# 回溯法模板

```cpp
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
```

# 回溯解决组合问题 1

回溯算法其实只有 **三个动作**：

```text
1 选择  
2 递归  
3 撤销选择
(任意顺序与指定顺序的差异在于)
```


```cpp
used[i] = true;  
path.push_back(i);  
  
backtrack(...);  
  
path.pop_back();  
used[i] = false;
```
