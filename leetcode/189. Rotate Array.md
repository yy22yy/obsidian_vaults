# 一、问题本质建模

长度为 `n` 的数组，向右轮转 `k` 次。

本质是：  
每个元素从 **原位置 i → 新位置 (i + k) % n**

这是**核心公式**，所有解法都围绕它展开。

---

# 二、基础预处理：k 的缩减

因为：

`k 可能远大于 n`

例如 `n=7, k=100`，实际上：

`100 % 7 = 2`

所以**第一步永远是：**

`k %= n;`

否则会超时 / 多余操作。

# 解法 4：三次反转法（最推荐 ⭐）

这是 **工程中最常用 & 最好记 & 最不容易写错的 O(1) 解法**。

---

## 1️⃣ 关键思想

设：

`nums = [A | B] A = 前 n-k 个 B = 后 k 个`

目标：

`[B | A]`

方法：

`整体反转 → 反转前 k → 反转后 n-k`

---

## 2️⃣ 举例

`[1 2 3 4 5 6 7], k=3`

### Step1：整体反转

`[7 6 5 4 3 2 1]`

### Step2：反转前 k 个

`[5 6 7 | 4 3 2 1]`

### Step3：反转后 n-k 个

`[5 6 7 | 1 2 3 4]`

完成！

---

## 3️⃣ 代码

```cpp
void reverse(vector<int>& nums, int l, int r) {
    while(l < r) {
        swap(nums[l++], nums[r--]);
    }
}

void rotate(vector<int>& nums, int k) {
    int n = nums.size();
    k %= n;
    reverse(nums, 0, n - 1);
    reverse(nums, 0, k - 1);
    reverse(nums, k, n - 1);
}

```
