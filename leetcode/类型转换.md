# 字符串string与数值类型的转换

## string--float

```cpp
float f1 = std::stof(s); // string → float 
double d1 = std::stod(s); // string → double
```

## int\long long --string

```cpp
int a = 42;
long long b = 1234567890123LL;

string s1 = to_string(a);   // "42"
string s2 = to_string(b);   // "1234567890123"
string s3 = to_string(-99); // "-99"
```
