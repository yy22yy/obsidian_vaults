# 字符串string与数值类型的转换

## 字符串转数字

```cpp
--记忆：string out float
	   string out double
	   
float f1 = stof(s); // string → float 
double d1 = stod(s); // string → double

	int  stoi
	long stol
```

## 数字转字符串

```cpp
int a = 42;
long long b = 1234567890123LL;

string s1 = to_string(a);   // "42"
string s2 = to_string(b);   // "1234567890123"
string s3 = to_string(-99); // "-99"
```
