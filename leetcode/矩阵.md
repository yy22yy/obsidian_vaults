# 数组创建矩阵

```cpp

vector<vector<int>> (嵌套向量)   m个n维向量

vector<int>(n * n) (一维化向量)  本质是单个向量，长度n*n

```

- **嵌套式：** `matrix[i][j]`。非常直观，符合数学直觉。
- **一维式：** `matrix[i * n + j]`。需要手动计算索引偏移量。


- **嵌套式：** 方便传递“某一行”。例如 `func(matrix[i])` 可以直接把第 $i$ 行当做一个普通的 `vector<int>` 传走。
- **一维式：** 传递某一行比较麻烦，通常需要使用 `std::span` 或迭代器。
