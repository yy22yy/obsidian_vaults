## 1. `matrix.size()`求的是什么？

对于 `std::vector<std::vector<int>> matrix`：

```cpp

matrix.size() = 行数  --- matrix 是由 m 行的 一维长度为n的向量组成 --大小为n*m
matrix[i].size() = 第 i 行的列数
```

# 矩阵旋转 90° 的倍数旋转 -- 两次旋转法

顺时针旋转 90° = 转置 --> 水平翻转每一行

```cpp
(i, j) → (j, n - 1 - i)

拆解为两步：

转置: (i,j) → (j,i)     交换行列
翻转: (j,i) → (j,n-1-i)  行内反转 -- 两个对称的行的下标和为 n-1


```

# 数学角度拆解这两步为什么等价于旋转

### 1️⃣ 转置：

`(i, j) → (j, i)`

### 2️⃣ 水平翻转：

`(j, i) → (j, n - 1 - i)`

合成：

`(i, j) → (j, n - 1 - i)`

这正是：

> **顺时针旋转 90° 的标准公式**
