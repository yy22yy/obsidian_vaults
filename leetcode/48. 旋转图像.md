## 1. `matrix.size()`求的是什么？

对于 `std::vector<std::vector<int>> matrix`：

```cpp

matrix.size() = 行数  --- matrix 是由 m 行的 一维长度为n的向量组成 --大小为n*m
matrix[i].size() = 第 i 行的列数
```

# 三、矩阵旋转 90° 的本质映射

顺时针旋转 90°：

```cpp
(i, j) → (j, n - 1 - i)
```

# 数学角度拆解这两步为什么等价于旋转

我们看坐标变换：

### 1️⃣ 转置：

`(i, j) → (j, i)`

### 2️⃣ 水平翻转：

`(j, i) → (j, n - 1 - i)`

合成：

`(i, j) → (j, n - 1 - i)`

这正是：

> **顺时针旋转 90° 的标准公式**
