# 一种高效的输出方法

例子：洛谷的超级玛丽题
P1000

```cpp
给定了一个十分不规则的字符串
如何进行高效的输出，不使用换行等等方法
```

### R"()" 的输出方法
```cpp
cout<<R"()"<<endl;

R"(字符串内容)"
R"(复制超级玛丽进入这里)"
```


- R表示这是一个原始字符串
- 括号 ( )中是实际的字符串内容
- 引号 "是字符串的边界
