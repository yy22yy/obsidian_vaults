[[ P1029 最大公约数和最小公倍数问题]]

# 手写版本
```cpp
int gcd(int a, int b) { return b ? gcd(b, a % b) : a; }
int lcm(int a, int b) { return a / gcd(a, b) * b; }
```

```cpp
最大公约数 -- gcd(a,b)
最小公倍数 -- lcm(a,b)  注意：lcm大于等于两个参数相乘的结果，一般赋予类型为      long long

===负数的使用 -- 本身会对参数取绝对值


gcd(-12, 8)   // → 4，等价于 gcd(12, 8)
lcm(-4, 6)    // → 12，等价于 lcm(4, 6)
gcd(-6, -9)   // → 3
lcm(-3, -5)   // → 15


```


# gcd的常见应用 -- 整除问题 互质问题

```cpp


// a 能被 b 整除 ↔ gcd(a,b) == b 
bool divisible = (gcd(a, b) == b); 
// a、b 互质 ↔ gcd(a,b) == 1 
bool coprime = (gcd(a, b) == 1); 

 求 a 的所有因子中，哪些也是 b 的因子（公因子） 
 → 等价于求 1..gcd(a,b) 中所有 gcd(a,b) 的因子
```
