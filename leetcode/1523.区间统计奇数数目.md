## goodcode

```cpp
class Solution { 
public int countOdds(int low, int high) 
{ return (high + 1) / 2 - low / 2; } 
}
```
## 原理说明

|表达式|含义|
|---|---|
|(high + 1) / 2|计算 [0, high] 中的奇数个数|
|low / 2|计算 [0, low-1] 中的奇数个数|
|两者相减|得到 [low, high] 中的奇数个数|
