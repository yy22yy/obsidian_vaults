
# 负数作为基数，基数不超过10

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,R;cin>>n>>R;
    //R是负进制数
    int m =n;//保存初始的n值
    string res= "";
    while(n!=0){
        // n = R * next + now
        int now = n % R;
        int next = n / R;
        if (now < 0){
            // n = R * (next+1) + now +abs(R)
            now += abs(R);
            next += 1;
        }
        n = next;
        res += now+'0';
    }
    reverse(res.begin(),res.end());
    if(res=="") res="0";
    cout <<m<<"="<<res<<"(base"<<R<<")";
}
```

- 方法：除基取余(基于修正)：保证余数始终为正数；
- 基于上面的方法，终止条件为：n等于0时，被除数和余数表示到尽头；
