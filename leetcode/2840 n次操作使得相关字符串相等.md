
# 分析

交换操作只能针对 j-i 是偶数的情况

- j-i是偶数的情况：两者同时为奇数或者偶数

可以统计所有同位置上出现的次数,本质上是一个统计问题


## 使用的统计对象


1. hash表 -- unordered _map <char,int>_
```cpp
unorder_map <char,int> e1,e2,o1,o2;

gammar: 
unorder_map <key,value> 
```

2. 四个一维数组

### 具体的统计实现

对象，等长的字符串s1和s2

- 初始的想法：使用两个for循环
```cpp
//统计两个字符串 偶数位 上的字符出现数
        for(int i=0;i<l1;i=i+2){
            //统计的实现
            even1[s1[i]]++;even2[s2[i]]++;
        }
        //统计两个字符串 奇数位 上的字符出现数
        for(int i=1;i<l1;i=i+2){
            //统计的实现
            odd1[s1[i]]++;odd2[s2[i]]++;
        }
```

### 统计后比较的实现

```cpp
bool result = (even1 == even2) && (odd1 == odd2);
```
