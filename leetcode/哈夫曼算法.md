# 2 `priority_queue`：优先队列（堆）

哈夫曼算法的核心结构。

头文件：

```cpp
#include <queue>
```

默认定义：

```cpp
priority_queue<int> pq;
```

特点：

```text
默认是最大堆
最大的元素在 top()
```

示例：

```cpp
pq.push(3);
pq.push(7);
pq.push(1);

cout << pq.top();   // 7
```

---

## 最小堆（哈夫曼需要）[[堆]]

哈夫曼要 **每次取最小元素**，定义：

```cpp
priority_queue<int, vector<int>, greater<int>> pq;
```

结构解释：

```text
priority_queue<
    数据类型,
    底层容器,
    比较函数
>
```

含义：

|部分|作用|
|---|---|
|`int`|元素类型|
|`vector<int>`|存储容器|
|`greater<int>`|小顶堆|

---

## 常用操作

|操作|代码|
|---|---|
|插入|`pq.push(x)`|
|访问最小|`pq.top()`|
|删除最小|`pq.pop()`|
|元素数量|`pq.size()`|
|是否空|`pq.empty()`|

示例：

```cpp
priority_queue<int, vector<int>, greater<int>> pq;

pq.push(4);
pq.push(1);
pq.push(7);

cout << pq.top();  // 1
```

# 3 哈夫曼算法使用流程

逻辑步骤：

```
1 把所有数放入最小堆
2 取两个最小元素
3 相加
4 加入答案
5 把和再放回堆
6 重复直到只剩一个
```

---

# 4 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){

    int n;
    cin >> n;

    priority_queue<int, vector<int>, greater<int>> pq;

    for(int i=0;i<n;i++){
        int x;
        cin >> x;
        pq.push(x);
    }

    int ans = 0;

    while(pq.size() > 1){

        int a = pq.top();
        pq.pop();

        int b = pq.top();
        pq.pop();

        int s = a + b;

        ans += s;

        pq.push(s);
    }

    cout << ans;
}
```

时间复杂度：

```
O(n log n)
```
