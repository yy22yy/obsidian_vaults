# 十进制转化为任何进制---除基取余

理来自：

$N=q×m+r$

其中

 块级公式
$$
r = N \bmod m
$$
$$
q = \left\lfloor \frac{N}{m} \right\rfloor
$$

r作为当前的位数存取
q替代为N的值进行后续的计算


# ==n进制转化为10进制==


```cpp
string s;
//n进制转化为10进制
    long long dec =0;
    for(char c : s){
        //n进制中当前字符所代表的位数
        int digit = isdigit(c) ? c -'0' :c -'A'+10;
        //从高到低做乘法即可--优美的写法
        dec = dec * n +digit;
    }
```

# ==10进制转化为m进制==

```cpp
//10进制--m进制
    string res = "";//定义一个空的字符串
    if(dec == 0){cout<<0;return 0;}
    while(dec>0){
        int r = dec%m;
        res += (r<10) ? r+'0': r -10 +'A';
        dec/=m;
    }
    //循环完之后对字符串进行反转
    reverse(res.begin(),res.end());
```
