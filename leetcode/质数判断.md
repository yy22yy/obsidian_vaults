自己构造一个类如下：

```cpp
bool isPrime(int n){
	//首先考虑非正整数情况
	if(n <= 1) return false;
	if(n == 2) return true;
	if(n % 2 == 0) return false;
	for(int i=3;i<n;i = i+2){
		if(n%i == 0) return false;
	}
	return true;
}
```

# 二、标准工程写法（√n 优化，最常用）

**数学依据**：  
如果 `n = a × b`，则必有 `a ≤ sqrt(n)` 或 `b ≤ sqrt(n)`。

所以只需要试除到 **√n**。

```cpp
bool isPrime(long long n) {  a
    if (n <= 1) return false;  a
    for (long long i = 2; i * i <= n; i++) {  a
        if (n % i == 0) return false;  a
    }  a
    return true;  a
}
```

**时间复杂度**：O(√n)  
这是 **算法竞赛 + 实际工程最常用方案**。
