# arr常见的查找算法[[注意事项]]

```cpp
// 最小值
*min_element(v.begin(), v.end());

// 最大值
*max_element(v.begin(), v.end());

// 查找
find(v.begin(), v.end(), x);

// 统计
count(v.begin(), v.end(), x);

// 二分查找
binary_search(v.begin(), v.end(), x);

// >=x
lower_bound(v.begin(), v.end(), x);

// >x
upper_bound(v.begin(), v.end(), x);
```
