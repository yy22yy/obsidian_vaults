
数字字符串的遍历问题

# 如何遍历 string s
## 1.下标遍历

```cpp
for (int i = 0; i < s.size(); i++) {  
    char c = s[i];  
}
```

**本质：**  
`string` 是 **连续存储的字符数组**，可以像数组一样用 `[]` 访问。

特点：

- 可用索引
    
- 可修改：`s[i] = '9';`

## 2. 范围 for（现代 C++ 推荐）

```cpp
for (char c : s) {  
}
```

**本质：**  
底层调用 `begin()` 和 `end()` 迭代器。

特点：

- 语义清晰
    
- 不能直接获取下标
    
- 只读时非常适合
## 3. 迭代器方式（理解 STL 必须）

```cpp
for (auto it = s.begin(); it != s.end(); it++) {  
    char c = *it;  
}
```

本质：

```cpp
iterator ≈ 指针  
*it ≈ 当前字符
```



# 如何将得到的string对象转化为int

## 1. 正确方式：`c - '0'`

```cpp
char c = '5';  
int x = c - '0';   // 5
```

### 原理（极其重要）

```text
ASCII码：

'0' -> 48  
'1' -> 49  
...  
'9' -> 57

因此：

'5' - '0' = 53 - 48 = 5

**结论：**

> 数字字符转整数，一律用：  
> 👉 `c - '0' 
```

# 三、字符串整体转 int 的现成函数

## 1. stoi（最常用）
```cpp
string s = "12345";  
int x = stoi(s);
```

支持：

- 正负号
    
- 自动处理前导空格



# 四、数字字符串逐位处理的典型场景

## 1. 求各位数字之和

```cpp
int sum = 0;  
for (char c : s) {
  
    sum += c - '0';
  
}
```

---

## 2. 判断是否为回文数字

```cpp
bool ok = true;  
for (int i = 0, j = s.size() - 1; i < j; i++, j--) {
  
    if (s[i] != s[j]) {
  
        ok = false;
  
        break;
  
    }
  
}

```


---


## 3. 构造大整数（超 long long）

```cpp
vector<int> digits;  
for (char c : s) {
    digits.push_back(c - '0');
}
```
