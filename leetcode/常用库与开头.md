# 固定开头


```cpp

cout<<fixed<<setprecision(3)<<a/b; 
cout<<fixed<<setprecision(需要精确的位数)<<a/b;
```

### 快速记忆表

|你想要的效果|必须写的顺序|备注|
|---|---|---|
|小数点后固定n位|fixed << setprecision(n)|必须先 fixed|
|不要科学计数法|加 fixed|否则大/很小的数会用 e 表示|
|改回整数风格（无小数点）|setprecision(0) + fixed|或者干脆不加 fixed|
|只影响当前输出|cout << setprecision(n) << 值;|但建议先设再输出|
