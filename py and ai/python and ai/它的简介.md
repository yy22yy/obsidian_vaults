# Claude Code 简介

  

## 什么时候需要运行 `/init`？

  

**只是简单聊天完全不需要运行 `/init`**。

  

`/init` 主要用于**软件开发场景**，当你需要：

- 探索或修改代码

- 调试问题

- 添加功能

- 查看项目结构

  

这时它才会派上用场。

  

---

  

## Claude Code 简介

  

**我不是 Claude Code** —— 我是 **Claude**（Anthropic 的 AI 助手），而 **Claude Code** 是一个命令行工具/应用，它让我能够在你的本地环境中：

  

- 📁 **读取和编辑文件**

- 🔍 **搜索代码库**（grep、文件查找）

- 💻 **执行 shell 命令**（运行测试、构建、git 操作等）

- 🌐 **访问网页**（获取文档、查 API 等）

- 🤖 **启动子代理**处理复杂任务

- 📋 **管理任务列表**跟踪进度

  

### 常用命令

  

| 命令 | 作用 |

|------|------|

| `/help` | 查看帮助 |

| `/init` | 初始化项目配置 |

| `/commit` | 创建 git 提交 |

| `/loop [间隔]` | 定时执行任务 |

| `!<命令>` | 直接执行 shell 命令 |

  

---

  

**总结**：现在这样聊天完全没问题，只有当你需要我帮你写代码、改 bug、查项目时才需要考虑 `/init`。
