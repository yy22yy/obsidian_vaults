介绍 .claude 这个目录下的东西

.claude 目录功能介绍

  📁 核心配置文件

  ┌───────────────┬──────────────────────────────────────────────────────┐
  │  文件/文件夹  │                       功能说明                       │
  ├───────────────┼──────────────────────────────────────────────────────┤
  │ settings.json │ 主配置文件 - 存储环境变量（API Token、模型、Base     │
  │               │ URL）                                                │
  ├───────────────┼──────────────────────────────────────────────────────┤
  │ models.json   │ 模型列表记录 - 跟踪已用模型的过期时间和顺序          │
  ├───────────────┼──────────────────────────────────────────────────────┤
  │ history.jsonl │ 对话历史记录                                         │
  └───────────────┴──────────────────────────────────────────────────────┘

  📁 数据缓存类

  ┌──────────────────┬──────────────────────────┐
  │      文件夹      │         功能说明         │
  ├──────────────────┼──────────────────────────┤
  │ cache/           │ 缓存数据（changelog 等） │
  ├──────────────────┼──────────────────────────┤
  │ paste-cache/     │ 图片/文件粘贴缓存        │
  ├──────────────────┼──────────────────────────┤
  │ file-history/    │ 文件修改历史版本         │
  ├──────────────────┼──────────────────────────┤
  │ shell-snapshots/ │ Shell 状态快照           │
  └──────────────────┴──────────────────────────┘

  📁 项目/会话类

  ┌───────────┬──────────────────────────────┐
  │  文件夹   │           功能说明           │
  ├───────────┼──────────────────────────────┤
  │ projects/ │ 每个项目独立的对话记录和状态 │
  ├───────────┼──────────────────────────────┤
  │ sessions/ │ 会话数据                     │
  ├───────────┼──────────────────────────────┤
  │ tasks/    │ 任务记录                     │
  ├───────────┼──────────────────────────────┤
  │ plans/    │ 计划模式的计划文件           │
  └───────────┴──────────────────────────────┘

  📁 插件/扩展类

  ┌──────────┬───────────────────────────────────────┐
  │  文件夹  │               功能说明                │
  ├──────────┼───────────────────────────────────────┤
  │ plugins/ │ MCP 插件（如 GitHub、Discord 等集成） │
  ├──────────┼───────────────────────────────────────┤
  │ ide/     │ IDE 集成相关                          │
  └──────────┴───────────────────────────────────────┘

  📁 其他

  ┌────────────┬────────────────────────────────────┐
  │   文件夹   │              功能说明              │
  ├────────────┼────────────────────────────────────┤
  │ backups/   │ 配置文件自动备份（每次修改前备份） │
  ├────────────┼────────────────────────────────────┤
  │ telemetry/ │ 使用遥测数据                       │
  ├────────────┼────────────────────────────────────┤
  │ debug/     │ 调试日志                           │
  ├────────────┼────────────────────────────────────┤
  │ downloads/ │ 下载的文件                         │
  └────────────┴────────────────────────────────────┘

  {
    "env": {
      "ANTHROPIC_AUTH_TOKEN": "sk-xxx",
      "ANTHROPIC_MODEL": "qwen3.5-flash",
      "ANTHROPIC_BASE_URL": "https://dashscope.aliyuncs.com/apps/anthropic"
    }
  }
