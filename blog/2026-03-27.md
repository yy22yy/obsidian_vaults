PowerShell 里用分号隔开：

```powershell
hexo clean; hexo generate; hexo deploy
```

## 关于把 LeetCode 仓库直接 copy 进去

可以，但有一个问题要提前知道：Hexo 会尝试把 `source/_posts/` 下的**所有** `.md` 文件都渲染成文章页面。LeetCode 仓库里的 Markdown 文件如果没有 frontmatter（就是开头那个 `---` 块），Hexo 会用文件名当标题，日期用文件修改时间，能跑但会很乱。

最稳的做法是先 copy 进去，跑一次 `hexo generate` 看看有没有报错，有报错再逐个处理。
